django创建表

2018-11-27 244 words
python
在settings.py配置好的前提下，打开models.py

创建基类(增加代码的复用)

#导包
from django.utils import timezone

#基类
class Base(models.Model):
    #创建时间 
    create_time = models.DateTimeField(default=timezone.now,null=True)

    class Meta:
        #能否被继承
        abstract = True

#用户类
class User(Base):
    #  用户名
    username = models.CharField(max_length=200)
    #  密码
    password = models.CharField(max_length=200)
    #  头像
    img = models.CharField(max_length=200)
    #分类 0普通用户 1会员
    type = models.IntegerField()
    #  主页
    num = models.IntegerField()
    #  联系方式
    phone = models.CharField(max_length=200)

    #  声明表名
    class Meta:
        db_table = 'user'

数据库迁移
python manage.py makemigrations myapp
python manage.py migrate

如果迁移不成功

  1. 删除myapp下的migrations
  2. 通过可视化工具Navicat进行物理删除，及删除数据库中的所有表（有时候删一次删不彻底，记得多重复几次物理删除操作）
  3. 重新建立数据库关系映射，迁移数据库
Mysql 用Navicat手动建表格

手动建表格，输入反向数据库入库命令

python manage.py inspectdb > mymodel.py
